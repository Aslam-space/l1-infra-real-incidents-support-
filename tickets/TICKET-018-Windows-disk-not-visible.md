# Ticket ID: TICKET-018

## Issue Title
New disk not visible in Windows system

## Description
User added new disk but cannot see it in File Explorer.

## User Reported Problem
Disk not showing after installation.

## Initial Checks (L1)
- Verified disk in BIOS
- Checked Disk Management

## Troubleshooting Performed
- Initialized disk
- Created partition
- Assigned drive letter

## Root Cause
Disk was not initialized.

## Resolution
Disk initialized and mounted successfully.

## Status
Resolved at L1 level

## SLA Impact
Resolved within SLA

## Learning
New disks must be initialized before use.
