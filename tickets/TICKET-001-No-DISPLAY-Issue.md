# Ticket ID: TICKET-001

## Issue Title
System powers ON but no display on monitor

## User Reported Problem
User reported that the CPU is powered on, fan spinning, but monitor shows no display.

## Initial Checks (L1)
- Checked monitor power source
- Verified HDMI/VGA cable connection
- Tried different monitor and port

## Troubleshooting Performed
- Reseated RAM modules
- Cleaned RAM slots
- Checked BIOS reset (Clear CMOS)
- Verified no external GPU issue

## Root Cause
Loose RAM connection caused system to fail POST.

## Resolution
Reseated RAM properly and system booted successfully with display.

## Status
Resolved at L1 level

## SLA Impact
Resolved within SLA time, no escalation required.
